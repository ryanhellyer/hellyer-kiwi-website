Get the total count of fans and followers from your social network profiles. Without the need of complicated API keys.

== Description ==

Simple yet powerful tool to show social networks statistic. It supports: Facebook, Twitter and Google Plus.

Allows to customize the colors and format of the counter using visual controls in the admin panel.

= Features: =
* Extremely easy set up. Copy/paste the URL of the social network. No use of API keys and tokens (more info below).
* Fully responsive, retina ready and compatible with mobile devices.
* Supports the 3 major social networks: Facebook, Twitter, Google+. More to come!
* Set the order of enabled social networks.
* Cache setting.
* Use it as a widget and as a shortcode.
* Practically null impact on page loads.
* Free updates and support.
* Import/Export settings to easily transfer configurations.
* For feature requests, please [contact us](https://www.prisna.net/contact-us/).

= API keys and tokens: =

This plugin does NOT require you to create any key or token to display the counter of your Facebook page, Twitter and Google Plus accounts.

The only information needed is the id, name or simply the URL of the social network, and that's it.

== Installation ==

1. Go to the WordPress admin panel.
1. Click *Plugins*, then *Add New*, then *Upload Plugin*.
1. Select the downloaded zip file, install it and activate it.
1. A new entry will appear under the *Plugins* main menu: *Social Counter*.
1. Set options at will.
1. Either go to the *Widgets* admin page, or use the shortcode on your pages, posts and any other WordPress resource.

== Frequently Asked Questions ==

= Will this plugin slow down my website? =
No, it won't. The plugin uses a smart cache system that stores count in the database and update it after the time set in the admin panel.
