== Description ==

The <a href="http://geek.ryanhellyer.net/products/code-comments/">PixoPoint Code Comments plugin</a> allows your site visitors to post fragments of code in their comments (wrapped in <code>...</code>) which are automatically encoded instead of them having to convert them to HTML entities such as &gt; and &lt;.

This plugin is based on code posted by Kaspars Dambis on his blog

For more information, please visit the <a href="http://geek.ryanhellyer.net/products/code-comments/">Code Comments plugin page</a>.

= Live Demo =

To see a live demo of the plugin in action, please visit <a href="http://geek.ryanhellyer.net/products/code-comments/#comment-864">http://geek.ryanhellyer.net/products/code-comments/#comment-864</a>.

== Installation ==

After you've downloaded and extracted the files:

1.  Upload the complete `code-comments` folder to the `/wp-content/plugins/` directory OR install it via the WordPress plugin repository browser<br />
2.  Activate the plugin through the 'Plugins' menu in WordPress<br />
