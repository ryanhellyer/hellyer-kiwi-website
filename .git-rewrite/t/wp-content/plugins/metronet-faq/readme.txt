== Description ==

Adds a simple FAQ section to your site, with pulldowns to reveal the answers. Content is added via a new post-type in the admin panel. Questions can be categorised into sections.

== Installation ==

* Install and activate the plugin
* Add the shortcode `[metronet-faq]` to the page or post you want the FAQ to appear on

= Optional =
* Use "section" optionally display only questions from a specific section, ie: `[metronet-faq section="popular question"]`. Sections can be set when editing a question.
* Modify your themes CSS to style the .the-answer section to make the answer stand out from the question. You can also style the .read-more and .faq-section classes for further design improvements.
